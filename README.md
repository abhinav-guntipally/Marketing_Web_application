# 🚀 Marketing Web Application – Cinematic Product Showcase

🌐 **Live Demo:**  
https://product-marketing-project.netlify.app/

---

# 📌 Project Overview

This project is a **frontend marketing website** built to demonstrate modern digital product presentation techniques.  
The goal of this application is to create an **interactive and visually appealing product showcase page** similar to marketing pages used by companies like Apple or Tesla.

The application focuses on delivering an engaging user experience through:

• Smooth scrolling sections  
• Interactive UI components  
• Product feature storytelling  
• Clean and modern design  

The project demonstrates how **React and modern frontend tools** can be used to build high-performance marketing websites.

# ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

git clone https://github.com/abhinav-guntipally/Marketing_Web_application.git

---

### 2️⃣ Navigate to the Project Folder

cd Marketing_Web_application/Web Application

---

### 3️⃣ Install Dependencies

npm install

---

### 4️⃣ Run Development Server

npm run dev

The application will run on:

http://localhost:5173

---

# 🚀 Build for Production

npm run build

Preview production build:

npm run preview
